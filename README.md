## Functionality
Real Time Weather App

## Tech Stack
Front-end: React<br>

## Frameworks
1. React
2. Tachyons


## Installation
1. Clone this repo
2. Run `npm install`
3. Change the URL in src/App.js to your own API backend.
4. Run `npm run dev`



## Frontend
Frontend Repo can be found [HERE](https://github.com/AdityaPote/mausam)
